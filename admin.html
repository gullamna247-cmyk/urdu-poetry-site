🚀 Setup Guide

Connect free services · $0/month

🏗️ Architecture
🌸
Vercel (Free)
Hosts frontend + API routes
100 GB/month
☁️
Cloudinary (Free)
Image CDN storage
25 GB bandwidth/month
🐘
Supabase (Free)
PostgreSQL database
500 MB · 50k req/day

Total cost: $0/month

📋 Step-by-Step
1

Create Supabase Project (Free)

Go to supabase.com → New Project → Free tier
Then: SQL Editor → New Query → paste the contents of supabase-schema.sql → Run

Copy from Settings → API:

Project URLhttps://xxxx.supabase.co
service_role keyeyJ… (secret, never expose to browser)
2

Create Cloudinary Account (Free)

Go to cloudinary.com → Free tier gives 25 GB storage + CDN
Copy from Dashboard: Cloud Name, API Key, API Secret

3

Generate bcrypt password hash

Run this locally once to get your hashed password:

node -e "const b=require('bcryptjs');console.log(b.hashSync('YOUR_STRONG_PASSWORD',12))"

The output (starting with $2b$) is your ADMIN_PASSWORD_HASH

4

Set Vercel Environment Variables

Vercel Dashboard → Your Project → Settings → Environment Variables

SUPABASE_URLhttps://xxxx.supabase.coRequired
SUPABASE_SERVICE_KEYeyJ… service_role keyRequired
CLOUDINARY_CLOUD_NAMEyour-cloud-nameRequired
CLOUDINARY_API_KEY123456789012345Required
CLOUDINARY_API_SECRETabcdef…Required
ADMIN_PASSWORD_HASH$2b$12$… (from step 3)Required
JWT_SECRETany random 32+ char stringRequired
5

Deploy to Vercel

npm i -g vercel vercel login vercel --prod

Or push to GitHub → connect repo in Vercel dashboard → auto-deploys on every push

✅

Done! Fully production-ready

• Upload from yourdomain.vercel.app/admin
• Images permanently on Cloudinary CDN
• Metadata forever in Supabase PostgreSQL
• Gallery auto-updates — no localStorage, no GitHub editing
